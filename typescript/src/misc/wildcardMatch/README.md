# Wildcard Match

Given an input string and a pattern, implement a wildcard matching function with support for literal characters `a` through `z` and the wildcard character `*` (which matches zero or more of any character).

**Example 1:**

```
Input string: "cat"
Input pattern: "c*t"
Output: true
```

**Example 2:**

```
Input string: "cat"
Input pattern: "*t"
Output: true
```

**Example 3:**

```
Input string: "dog"
Input pattern: "c*t"
Output: false
```

**Example 4:**

```
Input string: "cat"
Input pattern: "tac*"
Output: false
```
